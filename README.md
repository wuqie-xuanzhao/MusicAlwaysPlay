# 音乐一直放！

一个简单实用的工具，当其他程序播放声音时自动暂停音乐播放器，当没有其他声音时自动恢复音乐播放，让你的音乐一直放！

## 功能特点

- 自动检测系统中的音频播放状态
- 当检测到其他程序播放声音时，自动暂停音乐播放器
- 当其他程序停止播放声音时，自动恢复音乐播放
- 支持自定义音乐播放器进程名
- 支持自定义播放/暂停快捷键
- 美观的界面，支持亮色/暗色主题切换
